Getting Started

1

Installation

To begin using the AI-Enhanced Medical Dictionary:

  1. Ensure you have Python 3.8 or later installed on your system
  2. Install the required dependencies by running: pip install -r requirements.txt
  3. Launch the application by running: python main.py
2

Initial Setup

On first launch, the application will:

  • Create a default medical database with sample terms
  • Set up the necessary configuration files
  • Initialize the AI assistant with default parameters
Tip: For optimal AI performance, ensure you have a stable internet connection as some features require API access.

Interface Overview

1

Medical Dictionary Tab

The primary interface for searching and managing medical terms. Key components include:

  • Search Bar: Find terms by name, symptoms, or treatments
  • Category Filter: Narrow results by medical specialty
  • Online Search: Look up terms from authoritative medical sources
  • Term Display: View detailed information including definitions, symptoms, and treatments
  • Add Term Button: Create new medical term entries
Quick Tip: Press Enter after typing in the search bar to immediately search for terms.
2

AI Assistant Tab

Your interactive medical knowledge companion featuring:

  • Chat Interface: Ask medical questions in natural language
  • Predefined Prompts: Quick access to common medical queries
  • Context-Aware Responses: The AI remembers your conversation history
  • Term Explanations: Get detailed breakdowns of medical concepts
Note: While the AI assistant provides accurate information, it should not replace professional medical advice for diagnosis or treatment.
3

Analytics Tab

Visual insights into your medical database including:

  • Terms by Category: Bar chart showing distribution across specialties
  • Treatment Types: Pie chart of common treatment approaches
  • Prevalence: Horizontal bar chart of condition frequencies
  • Addition Timeline: Growth of your medical database over time

The analytics update automatically every 5 seconds or can be manually refreshed.

About Tab

Working with Medical Terms

Searching for Terms

1

To search for a medical term:

  1. Navigate to the Medical Dictionary tab
  2. Type your search query in the search bar
  3. Optionally select a category filter
  4. Press Enter or click the Search button

Adding New Terms

1

To add a new medical term:

  1. Click the "Add New Medical Term" button
  2. Fill in the term details (name, definition, symptoms, treatments)
  3. Select prevalence and category
  4. Click "OK" to save
Time Saver: Use the "Search Online" button to automatically populate fields from authoritative medical sources.

Getting AI Explanations

1

For detailed AI explanations of terms:

  1. Search for and select a term
  2. Click the "Get AI Explanation" button
  3. The application will switch to the AI Assistant tab with the explanation

Using the AI Assistant

The AI Assistant provides intelligent responses to medical queries. To use it effectively:

Basic Usage

1
  1. Navigate to the AI Assistant tab
  2. Type your question in the input field at the bottom
  3. Press Enter or click the Send button
  4. View the AI's response in the chat display

Predefined Prompts

Use the quick prompt buttons for common medical queries:

  • Explain like I'm 5: Simplified explanations of complex concepts
  • Latest treatments: Current treatment options for conditions
  • Differential diagnosis: Possible diagnoses based on symptoms
Important: The AI assistant uses GPT-3.5 by default. For more accurate responses, consider upgrading to GPT-4 in the configuration if available.

Understanding Analytics

The Analytics tab provides visual representations of your medical database:

Medical Terms by Category

Shows the distribution of terms across different medical specialties. Hover over bars to see exact counts.

Treatment Types Distribution

Displays the most common treatment approaches in your database as a percentage breakdown.

Prevalence of Conditions

Illustrates how common different conditions are in your database (Common, Uncommon, Rare).

Term Addition Timeline

Tracks the growth of your medical database over the current year.

Pro Tip: The analytics are automatically saved with your medical database and persist between sessions.

Troubleshooting

Common Issues

Problem: AI features not working
Solution: Check your internet connection and ensure you have a valid API key configured
Problem: Search returning no results
Solution: Try broadening your search terms or removing category filters
Problem: Application running slowly
Solution: Close other memory-intensive applications and ensure your system meets requirements

Data Recovery

Your medical database is automatically saved to medical_database.json. To back up your data:

  1. Close the application
  2. Make a copy of the medical_database.json file
  3. Store the copy in a safe location

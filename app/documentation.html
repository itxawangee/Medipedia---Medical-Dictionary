Medical Dictionary Application

Version 2.0 | AI-Enhanced Medical Reference Tool

Features

Medical Dictionary

  • Search medical terms by name, symptoms, or treatments
  • Filter results by medical categories
  • Online lookup from authoritative sources
  • Add custom terms to personal database

AI Assistant

  • Natural language medical queries
  • Detailed term explanations
  • Predefined question templates
  • Context-aware responses

Data Analytics

  • Visual term distribution by category
  • Treatment type analysis
  • Condition prevalence charts
  • Database growth timeline

Quick Start

Searching Terms

  1. Type search term in the search bar
  2. Press Enter or click Search button
  3. Use category filter to narrow results

Adding New Terms

  1. Click "Add New Medical Term"
  2. Complete all required fields
  3. Click OK to save
Tip: Use "Search Online" to automatically fetch term details from medical sources

Using AI Assistant

  1. Switch to AI Assistant tab
  2. Type your medical question
  3. Press Enter or click Send

Technical Information

Data Storage: Terms saved in medical_database.json

Requirements: Python 3.8+, PyQt6, OpenAI API key

Launch: Run python main.py

Note: For optimal performance, ensure stable internet connection when using AI features
